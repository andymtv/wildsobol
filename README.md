# Wild Sobol

First commercial website for Warsaw Renovation Company - WildSobol.

Stack of technologies i've used in this project:
  - HTML5.
  - CSS3 and SASS preprocessor.
  - Some Bootstrap elements (Modal Windows and Image slider).
  - A little bit of jQuery code.
  - Vanilla JS is used for creating some animation and for working with DOM-elements.
  - AJAX and PHP is used for sending email without reloading page.

You can visit website via link http://wildsobol.pl/
